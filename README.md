# Art Form Website 🎨

A simple and visually appealing website created to showcase different art forms, their cultural significance, and unique characteristics. This project was built using HTML, CSS, and JavaScript as part of my mini projects collection.

## 📌 Features
- Responsive and structured webpage layout
- Navigation bar for smooth section access
- Informative content about various art forms
- Images and visual elements for better engagement
- Clean and beginner-friendly UI

## 🛠️ Tech Stack
- **HTML5** – webpage structure and content organization
- **CSS3** – styling, layout, and design
- **JavaScript** – interactivity and dynamic behavior

## 📂 Project Structure
```bash
art-form/
│── index.html
│── style.css
│── script.js
│── images/
```

## 🚀 How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/Smrn-prydrshini/MiniProjects.git
   ```

2. Navigate to the project folder:
   ```bash
   cd MiniProjects/art\ form
   ```

3. Open `index.html` in your browser.
